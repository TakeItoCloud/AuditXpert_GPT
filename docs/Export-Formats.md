# Export Formats

## Supported formats
- JSON
- CSV
- Markdown
- HTML

## Intent
The export layer converts normalized findings into transport or review-friendly formats without changing the underlying finding shape.

## Format notes
- JSON preserves nested evidence and framework mapping objects most faithfully
- CSV uses a flattened representation for spreadsheet review
- Markdown provides lightweight technical sharing
- HTML provides a simple structured summary suitable for later reporting integration
