# AuditXpert Launcher Branding

## Direction
- Enterprise and assessment-focused
- Neutral, high-contrast palette
- Thin-client visual identity that does not compete with report output

## Messaging
- The desktop launcher is an operator shell.
- PowerShell modules remain the execution engine.
- AI features are optional and clearly labeled.

## Constraints
- No embedded secrets
- No hardcoded tenant information
- No visual implication that the launcher itself performs evidence collection
